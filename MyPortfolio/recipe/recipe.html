MP0 Tools:
This first tool was used to incorporate a transition where upon hovering over both my links and name on my landing page, it would change colors.

margin: 20px;
-webkit-transition: color 1s;
transition: color 0s;
}
a:hover {
color: #c7c8a4;
}
