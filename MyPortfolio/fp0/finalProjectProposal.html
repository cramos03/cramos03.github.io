Final Project Proposal

Intro

For my final project I plan on using P5 to create a paint-like drawing app, which lets the user download their art once they've completed their work. I've sketched out two drawings to mock-up a potential interface that I intend on embedding into the design for my application. These can be seen below:



As seen in my image above, I wanted to make it a two player game of Pong where I assigned the up and down arrow keys for the right paddle, while the W and S keys were assigned to the left paddle. Furthermore, my intention for this game was to have the ball launch in a random direction at the start to begin the match. I went with this option since I noticed that in the P5 library of Demos provided to us would land in a constant back and forth if neither player moved their paddle. My game is still under production since the ball does not continue moving once the game has begun and the paddles don't allow for repsonses when my keys are pressed.

Prototype Version

When considering a minimal version that demonstrates the core features of my app, I'd like to implement a brush (aka the mouse) with an attached color and brush stroke/size.

For my overall layout, I'll first aim to be able to draw and erase work on a canvas space. I'll attach a pencil and eraser icon to the side to distinguish the tool that is being used from one another. If I have a bit more time I'll shoot towards implementing some sort of color selector in the form of a color wheel for an easy visualization as well.

Final Version

What I envision for my final version builds off of my prototype. I'll maintain the color wheel as a color selector, but will also implement a storke/brush size selector in the form of a slider. This will help with adding detailed art to user's project rather than using the same brush size for the entire thing. If I have even more time, I'll experiement with adding different brush styles. This can be in the form of fade or blur tools, or even the brush shape itself.

Step-by-Step Plan

Below is the step-by-step plan for my development process when creating my project:

  1. Review class activity for creating continuous circles that will act as my brush
  2. Assign circle icon for changing brush color when clicked (do the same for eraser with matching background color)
  3. Use download example shared with Hannah to download art (saveCanvas tool)
    4. Completed Milestone 1: Prototype Version
  4. Create tool bar on the side of my canvas
  5. Create color wheel through P5 example and save color code to be used for brush color
  6. Set up slider for circle size (aka brush size)
  7. Update icon for eraser tool
  8. Add labels to each tool
    9. Completed Milestone 2: Final Version
